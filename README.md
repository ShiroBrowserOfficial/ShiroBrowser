# ShiroBrowser

Electronベースで構築された、次世代の高機能・多機能ブラウザです。
直感的な操作感と、ユーザーの自由を最優先した設計を実現しました。

## 主な特徴
- **フル・タブブラウジング**: 「+新しいタブ」で好きなだけページを開けます。
- **自由なホームページ**: 起動時や新タブ作成時、常に「君の選んだページ」から始まります。
- **スマートアドレスバー**: URL入力だけでなく、Google/Bing/DuckDuckGoでの直接検索もスムーズ。

## 使用方法
1. **起動**: アプリケーションを実行します。
2. **アクセス**: 画面上部のアドレスバーに URL（例: `https://google.com`）を入力して Enter。
3. **タブ操作**: 「+」ボタンで新しいタブを作成、「×」ボタンで閉じます。
4. **ナビゲーション**: 「←（戻る）」「→（進む）」「⟳（再読み込み）」で快適操作。

## 最小要件
| 項目 | 要件 |
| :--- | :--- |
| **OS** | Windows 10 / 11(64bit)専用 |
| **CPU** | 64-bit プロセッサ (Intel / AMD) |
| **メモリ** | 4GB RAM 以上（推奨: 8GB） |
| **ストレージ** | 350MB以上の空き容量 |
| **ネットワーク** | インターネット接続必須 |
| **グラフィック** | OpenGL 2.0 以上対応のGPU |
